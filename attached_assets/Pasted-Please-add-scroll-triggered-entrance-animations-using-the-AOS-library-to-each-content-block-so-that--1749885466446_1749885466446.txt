Please add scroll-triggered entrance animations using the AOS library to each content block so that text and images fade or slide into view gracefully as the user scrolls down the page.”

✅ 1. Install & Initialize AOS
Terminal:
bash
Copy
Edit
npm install aos
App.jsx or main.jsx:
js
Copy
Edit
import AOS from 'aos';
import 'aos/dist/aos.css';

useEffect(() => {
  AOS.init({
    duration: 800,
    easing: 'ease-in-out',
    once: true,
    offset: 100,
  });
}, []);
✅ 2. Apply data-aos Attributes to These Sections
Here are specific blocks from the page with recommended animation styles:

Section	HTML Tag	What to Add
Hero Text (Luxury Bridal Hair...)	<div>	data-aos="fade-up"
Pink CTA Button (“Our Elite Team of Artists”)	<button> or wrapper div	data-aos="fade-in"
Hollie DeSimanis Bio	<div>	data-aos="fade-right"
Cedar Sage Symans Bio	<div>	data-aos="fade-left"
Our Services Heading	<div> or <h2>	data-aos="fade-up"
Service Cards (grid)	Each card <div>	data-aos="zoom-in" and use data-aos-delay="100" etc for staggered animation
Our Portfolio Heading	<h2> or section wrapper	data-aos="fade-up"
Portfolio Images	Each image block	data-aos="zoom-in-up" or fade-up
Testimonials Cards	<div>	data-aos="fade-up"
Form Section (“Let’s Design Your Dream Bridal Look”)	Form container <div>	data-aos="fade-up"

✅ 4. Developer Notes
“Please don’t animate navigation bars, form fields, or footer elements. Only animate sections that enter the viewport as the user scrolls. Use fade-up, zoom-in, or fade-left/right to add graceful movement without distracting from the content.”
