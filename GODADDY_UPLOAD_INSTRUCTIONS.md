## How to Upload to GoDaddy

### Step 1: Access File Manager
1. Log into your GoDaddy account
2. Go to "My Products" → "Web Hosting"
3. Click "Manage" next to your hosting plan
4. Open "File Manager" in cPanel

### Step 2: Upload Files
1. Navigate to `public_html` folder
2. **Delete existing files** in public_html (if any)
3. Upload the extracted files from your download
4. Make sure `index.html` is in the root of public_html

### Step 3: File Permissions
- All files should be automatically set correctly
- If needed, set folders to 755 and files to 644
